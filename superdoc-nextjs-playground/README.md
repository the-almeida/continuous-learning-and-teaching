# Next.js + SuperDoc Playground

> **Status: Finished** — This project is complete. It lives inside the `continuous-learning-and-teaching` monorepo as a finished learning exercise.

This application is a playground to try out [SuperDoc](https://www.superdoc.dev) with [Next.js](https://nextjs.org). The first goal is to display my CV and experiment with SuperDoc configs.

**Live demo:** https://superdoc.thealmeida.online/

## Tech Stack

- [Next.js](https://nextjs.org) (App Router)
- [React](https://react.dev)
- TypeScript
- [SuperDoc](https://www.superdoc.dev) — open source document editor

## Running Locally

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the CV.
